# Product Requirements Document

## Product
ArchonOS Next

## Target User
Windows users seeking local-first AI workflows, knowledge management, and persistent memory.

## Core Features
- Local Knowledge Base
- Document Ingestion
- Semantic Search
- Workflow Engine
- Persistent Memory
- Provider Abstraction

## Non-Goals
- Build a new LLM
- Depend on a single runtime
- Require cloud infrastructure

## Alpha Success Criteria
A user can install the system, ingest documents, search knowledge, run workflows, and preserve memory on a single laptop.
