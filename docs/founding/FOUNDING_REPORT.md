# ArchonOS Next Founding Report

## Mission
Build a local-first AI operating system focused on knowledge, workflows, memory, and automation.

## Core Principles
- Local First
- Knowledge First
- Workflow First
- Runtime Agnostic
- Provider Agnostic

## Target Platform
Windows 11 + WSL2
16GB minimum RAM
32GB recommended RAM

## Default Stack
SQLite, Python, OpenRouter.

## Initial Milestone
Ingest documents, build local knowledge base, search, run workflows, persist memory.
